# 📱 APK 빌드 및 설치 안내 가이드

나만의 사전 앱을 모바일에 설치하기 위한 소스코드 준비가 완료되었습니다. 아래 절차를 따라하시면 자동으로 APK가 생성됩니다.

## 1. GitHub 저장소에 코드 업로드
현재 작업된 코드(scratch 폴더 내의 모든 파일)를 본인의 **GitHub 저장소**에 업로드(Push)해 주세요.
> [!NOTE]
> 저장소 이름은 무엇이든 상관없습니다.

## 2. 자동 빌드 시작
코드가 업로드되면 `.github/workflows/build.yml` 설정에 의해 GitHub Actions가 자동으로 빌드를 시작합니다.
- GitHub 저장소 상단의 **[Actions]** 탭을 클릭하면 진행 상황을 볼 수 있습니다.

## 3. APK 다운로드
빌드가 완료(초록색 체크 표시)되면 해당 워크플로우 실행 항목을 클릭합니다.
- 화면 하단의 **Artifacts** 섹션에 있는 `dictionary-app-debug` 파일을 클릭하여 다운로드합니다.
- 압축을 풀면 `app-debug.apk` 파일이 나옵니다.

## 4. 폰에 설치
1. `app-debug.apk` 파일을 스마트폰으로 옮깁니다 (카카오톡 나에게 보내기, 메일 등).
2. 폰에서 파일을 실행하여 설치합니다.
   - "출처를 알 수 없는 앱 설치" 경고가 뜨면 **허용**을 선택해 주세요 (본인이 직접 만든 앱이므로 안전합니다).

> [!TIP]
> 앱 이름: **나만의 사전**
> 앱 아이콘: 기본 아이콘으로 생성됩니다 (추후 커스터마이징 가능).
